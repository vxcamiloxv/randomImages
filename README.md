RandomImage Test
===============

RandomImage test with Vue.js 2.0 and Node.js basic server

Install
----------

### Requirements ###
- Node.js v4.x or higher

```shell
  npm install
```

Usage
-------

```shell
  npm run serve
```

#### Config ####
Configuration file for server, by default take ./config.js
*see config.example.js*.

#### Build ####

Build assets and js client
```shell
  npm run build
```

#### Test ####

```shell
  npm run test
```
